Calculadora para el índice de nebulosidad

El índice de nebulosidad permite medir la dificultad de lectura de un texto. El índice de nebulosidad ideal es 15.

Inebulosidad = 0,4 * ( (Npalabras / Nfrases) + (Npalabras largas * 100 / Npalabras) )
